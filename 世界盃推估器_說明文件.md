# 2026 世界盃推估器 — 建構過程與模型說明

> 離線 HTML 應用程式（`index.html` ＋ 資料檔 `世界盃資料.js`，目前版本 **v4.1**）
> 對象：2026 美加墨世界盃 48 強
> 文件日期：2026-06-13

> **檔案結構（v3.9 起）**
> - `index.html` — 主程式（介面與模型邏輯）
> - `世界盃資料.js` — 所有靜態資料（48 隊評分/身價/年齡、賽程、球員、新聞、官方對陣樹…）。**要更新整批資料只改這個檔。**

> - 你的個人編輯（比分、新聞、設定）存在瀏覽器 localStorage，可在「⚙️ 設定 → 匯出備份」存成可攜的 `.json` 檔。

## 一、專案概述

這是一款**單一 HTML 檔案、可離線使用**的世界盃比賽推估器。核心理念是把「球隊長期實力」用十多項因子修正成「有效 Elo」，再透過機率模型推算單場勝負、比分，以及整屆冠軍機率。

**設計原則**

- **單檔離線**：所有資料內建，雙擊即用；連網時才更新比分與新聞，平常完全離線可跑。
- **透明可解釋**：每一場預測都能看到逐項因子的加減分，不是黑盒子。
- **可調校**：所有模型參數都在「⚙️ 設定」可調，使用者能依自己的判斷微調。
- **資料只存本機**：使用 localStorage，不上傳任何資料。
- **不接 AI**：第一版刻意不接大型語言模型（翻譯用免金鑰的公開端點），維持零成本、零相依。

### 建構方法論

1. **資料蒐集**：透過多來源網路檢索並交叉查核（FIFA、Wikipedia、ESPN、eloratings.net、Transfermarkt、RotoWire、Opta），把 48 隊的 Elo、身價、年齡、主力球員、PK 史、賽程等內建成靜態資料。
2. **逐步驗證**：每次改動都在本機伺服器預覽，用程式化檢查驗證數值正確、無錯誤，再交付。
3. **對標校準**：把模型輸出對照 Opta 超級電腦與博彩市場，找出系統性偏差再修正（見第五節）。
4. **證據化定參**：對「僑民主場」「高原」這類沒有現成係數的因子，動用了一個**多代理工作流**——先由研究代理查證據、再由對抗審查代理質疑「數值會不會太大／太小」，最後採用裁定值，避免拍腦袋。

## 三、推估模型詳解

### 整體流程

```
┌──────────┐   ┌─────────┐   ┌──────────┐   ┌──────────────┐   ┌──────────┐
│ 有效 Elo │ → │ 勝率 We │ → │ 期望進球 │ → │ Poisson 比分 │ → │ 蒙地卡羅 │
│ ±12 因子 │   │ Elo→機率│   │ λ 分兩隊 │   │  勝平負＋比分 │   │ 模擬數千次│
└──────────┘   └─────────┘   └──────────┘   └──────────────┘   └──────────┘
```

### 第一步：有效 Elo ＝ 基準 Elo ± 因子

每隊的「有效實力」是基準 Elo 加上以下各項修正的總和：

| 因子 | 算法 | 資料來源 |
|------|------|---------|
| **基準 Elo** | 1421（卡達）～ 2157（西班牙） | eloratings.net 世界足球 Elo（2026/6/11） |
| **市場身價** | 全隊身價取對數 → z-score 對齊 Elo 分布 → 混入 30% | Transfermarkt（學界最強單一預測變數） |
| **球員狀態** | 四檔：缺陣 −權重×8｜低迷帶傷 −50%｜巔峰 +25%（權重 1～5★） | 23 強隊主力名單＋ESPN 出賽名單比對 |
| **新聞因素** | 每則事件 ±Elo（可開關／調整） | ESPN 即時新聞自動判讀＋手動 |
| **賽中動態** | 每場真實比分以 K=40×進球差倍率調整 Elo | 實際表現 vs 賽前預期的落差 |
| **特質** | 韌性·團結 ±2.5/級、中場 ±3/級（總上限 ±15） | 主觀評分（民族性／團隊／中場控制） |
| **年齡結構** | >28.5 歲每歲 −8、<25.5 歲每歲 −6（上限 −15） | RotoWire 26 人最終名單平均年齡 |
| **主場優勢** | 主辦國在自家場館 +50 | 美／加／墨 |
| **高原** | 海拔 1200m 以上每千米約 43（Azteca ≈ −45）；墨／厄／哥免疫 | McSharry BMJ 2007，經移地訓練折減 |
| **僑民主場** | 拉美隊在美場館墨西哥 +24（他隊按權重、加拿大半額） | 幽靈球賽研究：主場優勢約半數來自觀眾 |
| **休息差／手動** | 休息每天差 8（上限 24）；手動任意加減 | 淘汰賽休息天數＋使用者判斷 |

> **設計重點：避免重複計分。** 市場身價、特質、年齡這些因子，效果方向是去**修正** Elo 沒抓到的資訊，而不是重複 Elo 已含的實力。例如阿根廷 Elo 高（第 2）但陣容老化、身價僅第 8 → 身價與年齡因子把它往下拉；英格蘭身價世界第二但 Elo 第四 → 往上拉。

#### 兩個關鍵因子的證據基礎

**高原（線性 + 門檻）**
McSharry 在 BMJ（2007，1,460 場南美國際賽）發現每 1,000 公尺海拔差影響約 0.5 球。但那是球員飛來飛去、零適應的極端情境；世界盃有賽前移地訓練，效果打折。經對抗審查後採「海拔 1200m 以上每千米 43 Elo」：

- 墨西哥城 Azteca（2,240m）：約 −45 Elo
- 瓜達拉哈拉（1,560m）：約 −15 Elo（落在「2010 世界盃實證零效應」的低海拔區間，刻意壓低）
- 蒙特雷（540m）及所有美／加場館：0

**僑民主場**
幽靈球賽（COVID 無觀眾）的自然實驗顯示，主場優勢**約一半來自觀眾**（影響裁判＋情緒支持），另一半來自旅行／熟悉場地——而後者是拉美隊在美國中立場館**得不到**的。墨西哥在美國 NFL 球場場均 5～7 萬綠衫球迷，形同主場。採墨西哥 +24，其餘按權重：

| 球隊 | 權重 | 美國場館加成 |
|------|------|------|
| 墨西哥 | 1.0 | +24 |
| 阿根廷 | 0.7 | +17 |
| 哥倫比亞 | 0.6 | +14 |
| 巴西 | 0.5 | +12 |
| 厄瓜多 | 0.35 | +8 |
| 烏拉圭 | 0.18 | +4 |
| 巴拉圭 | 0.1 | +2 |

（加拿大場館減半；自家主辦國不與主場優勢疊加。）

### 第二步：Elo 差距 → 勝負機率與比分

**勝率公式**
```
We = 1 / (1 + 10^(−ΔElo / 600))
```
兩隊有效 Elo 相減得 ΔElo。分母用 **600**（標準 Elo 為 400），讓單場勝率更平緩、更貼近博彩市場、避免過度自信。

**期望進球（含攻防節奏，v4.1）**
```
λ_左 = 2.6 × We × 節奏^γ      λ_右 = 2.6 × (1 − We) × 節奏^γ
節奏 = (att_左 × def_右 + att_右 × def_左) / 2
```
總進球 2.6 按勝率比例分配，再乘「攻防節奏」。att/def 由各隊近年場均進球/失球正規化而來（均值 1.0）；攻擊型遇上漏勺防線 → 節奏 >1（高比分），兩支鐵桶 → 節奏 <1（悶平）。**關鍵設計**：兩隊同乘節奏，只放大總進球規模、不改變勝率比例——因為「誰贏」已由 Elo＋身價等強信號決定，若讓攻防再去傾斜勝負會與實力雙重計分（實測會讓西班牙冠軍率從 16% 暴衝到 25%，跑掉校準）。γ 預設 0.5，可調。

**Poisson 比分分布＋動態平手（v4.1）**
```
P(i:j) ∝ Poisson(λ_左, i) × Poisson(λ_右, j) × ρ^(i=j)
ρ = 1 + (1.2 − 1) × exp(−|ΔElo| / 350)
```
獨立 Poisson 在學界公認**系統性低估平局**，故對角線乘平手修正。但這個低估在**勢均力敵時最嚴重**，故加成隨實力差**淡出**：兩隊接近時給滿 1.2、一面倒時趨近 1.0（那種平手本來就罕見，不該再灌）。

**淘汰賽 PK（v4.1，改 70/20/10）**
```
晉級率 = 70% × 當前實力 + 20% × 門將品質 + 10% × PK大戰史
```
90 分鐘平手後混合判定。**為何下修 PK 史權重**：原本各半（50/50）太重——「德國 4-0」是橫跨數十年、不同球員的 4 場樣本，當 50% 等於放大雜訊。改成主要看當前實力、輔以門將品質（用名單門將星級）、PK 史只佔一成。PK 史採真實紀錄（德 4-0、克 4-0、阿 6-1、英 1-3、西 1-4…），拉普拉斯平滑。

### 第三步：蒙地卡羅整屆模擬

- **逐場抽樣**：未踢的小組賽依比分分布隨機抽一個比分；已踢的直接採真實結果。
- **538 式擾動**：每次模擬給各隊隨機浮動一個狀態（σ = 45 Elo 的高斯雜訊），模擬「狀態起伏」，避免「強隊必勝」的過度自信——這是 FiveThirtyEight 的招牌做法。
- **官方對陣樹**：淘汰賽採 FIFA 官方 M73–M104 槽位定義（如 M79 ＝ A 組第一 vs 指定組別之第三），逐場依官方場館計算主場／高原／僑民效應。
- **統計**：跑數千次後，計算每隊走到 16 強／8 強／4 強／決賽／奪冠的次數百分比。
- **官方鎖定**：小組賽全部踢完後按「一鍵更新」，會自動抓官方真實對陣與賽果逐場鎖定，模擬只跑還沒踢的部分。

## 五、校準與專家對標

模型最初對西班牙、阿根廷**過度自信**（兩隊合計冠軍機率約 60%），與所有專家模型不符。經多輪校準後收斂到合理區間：

| 球隊 | 校準前 | v3.4 | v3.5 | v3.6 後 | **Opta** | **博彩市場** |
|------|-------|------|------|--------|---------|------|
| 西班牙 | 31.9% | 21.0% | 16.8% | ~16% | 16.1% | 18.2% |
| 阿根廷 | 27.6% | 20.0% | 16.3% | ~17% | 10.4% | 10.0% |
| 法國 | 12.6% | 10.8% | 11.5% | ~11% | 13.0% | 16.7% |
| 英格蘭 | 5.9% | 5.9% | 7.0% | ~7% | 11.2% | 12.5% |
| 葡萄牙 | 5.1% | — | 7.0% | ~7% | 7.0% | 10.0% |

**校準的四個槓桿**

1. **特質係數減半 + 上限 ±15**：修正質化因子與 Elo 的重複計分（阿根廷特質從 +26 降到 +13）。
2. **市場身價支柱**：把英格蘭往上拉、阿根廷往下壓，方向正確。
3. **實力差敏感度 400 → 600**：單場勝率變平緩，六輪淘汰賽複利下來，冠軍機率從過熱回到合理。
4. **538 式模擬擾動**：注入不確定性，避免過度自信。

> **誠實註記**：阿根廷在本模型仍略高於 Opta，主因是我們**明確納入了僑民主場效應**（阿根廷在美國場館享有偏袒性球迷氛圍），這是 Opta 那類模型未必明確編碼的因子。這不是錯誤，而是建模選擇的差異。

## 七、誠實的限制

- **Elo 是長期實力**，單場爆冷只能靠輸入比分與新聞修正——這正是賽中動態與新聞分頁的用途。
- **特質、僑民主場、PK 體質是估計值**，非硬數據；可在設定調整或關閉。
- **新聞為機器翻譯**，偶有誤譯（曾把「戰平 tie」譯成「領帶」）；影響值判讀以英文原文為準。
- **比分來源 ESPN 為非官方介面**，改版時可能失效（App 本體不受影響）。
- **平手機率**經修正仍可能略低於真實；**第三名分配**在官方表多解時可能與正式結果略異。
- **這是娛樂與學習用的推估工具，不是投注建議。**
